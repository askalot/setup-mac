# Git

## Install Git

```
brew install git
```

## Add config

Copy `https://raw.githubusercontent.com/askalot/dotfiles/master/gitconfig` to `~/.gitconfig`.

## Install gitsh

```
brew tap thoughtbot/formulae && brew install gitsh
```

## Install GitUp

```
brew cask install gitup
```

## Install Sourcetree

```
brew cask install sourcetree
```
